# Job Market Analysis Dashboard (SQL + Power BI)

## Project Overview
This project analyzes job market trends using SQL and Power BI. The objective is to identify in-demand job roles, required skills, salary trends, industry opportunities, and hiring patterns across companies.

## Objectives
- Analyze job demand across industries
- Identify the most in-demand skills
- Explore salary trends by job role and industry
- Evaluate company hiring patterns
- Build interactive dashboards for decision-making

## Tools Used
- MySQL
- SQL
- Power BI
- Power Query
- DAX
- Excel

## Dataset Summary
- Total Jobs: 742
- Total Companies: 343
- Total Industries: 60

## Key Insights
- Data Scientist is the most demanded role.
- Python is the most in-demand skill.
- SQL and Excel remain highly demanded skills.
- Other Retail Stores offer the highest average salary.
- Takeda Pharmaceuticals is among the top hiring companies.
- Biotech & Pharmaceutical industry has the highest job demand.
